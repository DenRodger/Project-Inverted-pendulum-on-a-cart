# Project-Inverted-pendulum-on-a-cart
Project for Simulation of technical system dynamics
**Мета**: Змоделювати систему «перевернутий маятник на візку» (cart-pole). Синтезувати PID-регулятор для стабілізації. Дослідити область притягання та робастність до збурень.

**1.1. Об’єкт моделювання**
Візок масою M рухається по горизонтальній рейці під дією сили F. На візку закріплено стержень (маса m, довжина l) на шарнірі. Система описується ОДУ 4-го порядку: позиція візка x, швидкість візка dx/dt, кут маятника θ, кутова швидкість dθ/dt. Нелінійність: sin(θ), cos(θ) у рівняннях Лагранжа, обмеження ходу візка (сатурація).
